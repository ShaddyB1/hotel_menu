How to Update Menu:

  • Go to your GitHub repository
  • Click on menu.html
  • Click the pencil icon to edit
  • Make your changes
  • Click "Commit changes"
  • Wait 1-2 minutes for changes to update
  • Refresh this page to see changes

To Save QR Code:

  • Right-click on the QR code
  • Select "Save image as..."
  • Choose save location
  • Print and display the QR code for your guests
